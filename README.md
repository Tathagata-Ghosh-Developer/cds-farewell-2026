## ✨ Features

| Feature | Description |
|---|---|
| 🖥️ **Loading Screen** | Terminal-style boot sequence with cycling ML jokes |
| 🚪 **Gatekeeper** | Choose Senior (👑) or Junior (🤡) — each gets a different experience |
| 👑 **Senior VIP** | Red carpet treatment, emoji rain, floating meme quotes, VIP ticket reveal |
| 🤡 **Junior Broke Zone** | UPI QR code (₹2,000), payment guilt-trip, CUDA Out of Memory panic button |
| 📍 **Event Details** | Neon cyberpunk cards with date, time, venue, dress code |
| 📸 **Photo Carousel** | Draggable Polaroid-framed photos with funny captions |
| 🔊 **Meme Audio** | 7 contextual sound effects triggered by interactions |
| 🥚 **Easter Eggs** | 3-click venue reveal, double-click popups, random chaos |
| 📱 **Fully Responsive** | Works on phones, tablets, laptops, and big screens |
